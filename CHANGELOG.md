# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-05

### Added
- Initial release with 46 tools covering complete Early API v4
- User & Authentication: `early_get_me`, `early_list_users`
- Time Tracking: start, stop, edit, cancel, get current tracking
- Time Entries: list, get, create, edit, delete
- Activities: list, create, edit, archive, unarchive
- Folders: full CRUD + member management (10 tools)
- Tags: list, create, update, delete
- Mentions: create, update, delete
- Leaves/Time Off: create, approve, deny, delete (5 tools)
- Webhooks: list events, subscribe, unsubscribe (5 tools)
- Reports: generate report, today summary
